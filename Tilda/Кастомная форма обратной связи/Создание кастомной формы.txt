Для создания кастомной формы обратной связи нужно.

1) Создать фальшивую форму, из элементов в Zero Blocks, используя формы для ввода текста и отправки, маскеруя кнопки этих форм в Zero Blocks.
   Изображения  в фальшивой форме могут иметь функцию радио кнопок, галочек проще говоря. Типо  пользователь выбирает какой товар ему больше нравится  при нажатии на картинку.
   Обязательно укажите для всех полей ффода имя - name="" в настройках input в Zero Block.
2) Cоздание настоящей формы, мы создаём те элементы формы которые мы создали в фальшивой форме ,они проще говоря должны повторятся. Если вы создали в фальшивой форме , изображения то
   для выбора . То в настоящей форме создайте радио кнопки.
3) После создания фальшивой формы и настоящей, настоящяя форма обратной связи будет спрятана от пользователя и то что мы будем ввводить  в фальшивой форе будет отображатся и в настоящей.
   Для этого нам поможет этот скрипт со стилями:
<style>

/*скрываем блок с настоящей формой*/

#rec635972210, .t-submit {

display:none;

}

/*скрываем кнопки фейковых форм*/

#rec635577794 .t-submit {

display:none !important;

}

</style>

<script>

$('.класс кнопки фейк формы').click(function() {

$('textarea[name="имя_настоящей_формы"]').val($('textarea[name="имя_фейк_формы"]').val()); //текстовое поле

$('select[name="имя_настоящей_формы"]').val($('select[name="имя_фейк_формы"]').val()); //выпадающий список

$('input[name="имя_настоящей_формы"]').val($('input[name="имя_фейк_формы"]').val()); //инпут (поле в одну линию)

$('#rec635972210 .t-submit').trigger('click'); //клик по кнопке "настоящей" формы

});

});

</script>
